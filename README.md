# Crypto Price Tracker (GitHub Pages Project)

This project was created as part of a GitHub Pages assignment.

## 📌 Project Overview

This website includes:

1. A static table showing Bitcoin and Ethereum prices.
2. A Markdown report documenting how the prices were collected.
3. A live version that fetches real-time prices from the CoinGecko API.

---

## 🌐 Website

GitHub Pages URL:

https://jeon-dahye.github.io/

---

## 📂 Files in This Repository

- `index.html` → Main homepage
- `prices.md` → Markdown documentation of static prices
- `prices.html` → HTML version converted from Markdown
- `prices-live.html` → Live price tracker using CoinGecko API

---

## 🔄 Static vs Dynamic Version

### Static Version
- Prices were manually collected from CoinGecko.
- Documented in Markdown format.
- Converted into HTML.

### Live Version
- Uses CoinGecko public API.
- Automatically refreshes every 10 seconds.
- Displays the latest available market data.
